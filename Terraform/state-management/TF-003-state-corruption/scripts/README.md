# Scripts

This directory contains helper scripts for detecting, validating, backing up, and recovering Terraform state.

## Important

Review all scripts before execution.

These scripts assist investigation and recovery.
They do not replace manual validation or incident response procedures.

## Scripts

* detect-state-corruption.sh → detect corruption indicators
* backup-state.sh → create emergency state backup
* restore-state.sh → assist backup restoration workflow
* validate-state.sh → validate state health
* state-health-report.sh → generate state health report
