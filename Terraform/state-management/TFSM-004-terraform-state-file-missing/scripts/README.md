# Scripts

This directory contains helper scripts for detecting, restoring, validating, and reporting Terraform state recovery.

## Important

Review all scripts before execution.

Scripts assist incident response.
They do not replace manual validation or recovery procedures.

## Scripts

* detect-missing-state.sh → detect missing state
* restore-state.sh → restore state backup
* validate-state.sh → validate state health
* state-recovery-report.sh → generate recovery report
